This is a classic Snake Game implemented in Python using the turtle GUI. The game is about controlling a snake to collecting food while avoiding collisions with the walls or its tail.
Features:
Gameplay: Control the snake using arrow keys to navigate through screen and consume food.
Score Tracking: Keep track of the score based on the number of food items collected.
Collision Detection: Game is over when the snake collides with the wall or itself.
User-friendly Graphics: Simple and clean graphics using the turtle GUI.
